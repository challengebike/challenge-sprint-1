Banco de Dados

Um banco de dados é uma coleção organizada de informações - ou dados - estruturadas, normalmente armazenadas eletronicamente em um sistema de computador.

Como proposta, pretendemos desenvolver um banco de dados para gerenciar as informações de todo o projeto: armazenar os dados do cliente e da bike, armazenar as fotos e vídeos enviados pelos clientes no momento da vistoria e os demais dados envolvidos no processo.
